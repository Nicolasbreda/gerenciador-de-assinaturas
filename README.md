# Gerenciador de Assinaturas

Aplicação de terminal desenvolvida em Java para gerenciamento de assinaturas recorrentes. O sistema permite cadastrar serviços por assinatura, acompanhar próximas cobranças, calcular gastos mensais e gerar relatórios simples para auxiliar no controle financeiro.

## Funcionalidades

- Cadastro de assinaturas
- Edição de assinaturas
- Cancelamento de assinaturas
- Listagem de assinaturas ativas e canceladas
- Consulta das próximas cobranças
- Cálculo do gasto mensal
- Geração de relatórios financeiros

## Tecnologias utilizadas

- Java 21
- Collections Framework
- LocalDate (Java Time API)
- Programação Orientada a Objetos (POO)
